# Button

With `MDX`, we can define a story for `Checkbox` right in the middle of our
Markdown documentation.

export const Template = (args) => <Button {...args} />

<Canvas>
<Story 
  name="Main Story"
  args={{
    label: 'Button',
    size: 'md'
}}>
  {Template.bind({})}
</Story>
</Canvas>

<Canvas>
<Story 
  name="Green"
  args={{
    accentColor: 'forestgreen',
    label: 'Green Button',
    size: 'md'
}}>
  {Template.bind({})}
</Story>
</Canvas>
