# Checkbox

With `MDX`, we can define a story for `Checkbox` right in the middle of our
Markdown documentation.

export const Template = (args) => <Button {...args} />

<Canvas>
<Story 
  name="positive"
  args={{
    status: 'Positive',
    label: 'Positive'
}}>
  {Template.bind({})}
</Story>
</Canvas>
