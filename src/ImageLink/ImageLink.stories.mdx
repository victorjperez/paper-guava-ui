# ImageLink

`ImageLink` is an anchor component used to display child elements overtop an image.

export const Template = (args) => <ImageLink {...args} />

<Canvas>
<Story 
  name="Main Story"
  args={{
    href:"/writing",
    imgSrc: "https://victorjperez.github.io/writing/assets/img/morgantown.jpg",
    children: <p>This Town</p>
  }}>
  {Template.bind({})}
</Story>
</Canvas>

## Component Options

Below is a list of props for `ImageLink` and their usage.

### Image Brightness

You can set the Brightness of the image by passing a `brightness` value.

```jsx
<ImageLink brightness="90%" />
```
